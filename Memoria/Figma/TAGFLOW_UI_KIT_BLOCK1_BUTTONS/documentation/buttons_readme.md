
TagFlow UI Kit - Buttons
========================

This folder contains the button components for TagFlow in both light and dark themes.
Files:
- svg/ (vector editable)
- png/ (high-resolution previews)

Variants included:
- primary_default, primary_hover, primary_pressed, primary_disabled
- primary_icon_left, primary_icon_right
- secondary_default, secondary_disabled, ghost_default

Tokens used: tokens/design_tokens_light.json and tokens/design_tokens_dark.json
AutoLayout guidance:
- Height: 56px (iPhone 14 Pro scale)
- Horizontal padding: 20px
- Vertical padding: 12px
- Corner radius: 14px
- Font: Inter 16px semibold

Usage:
- Import SVGs into Figma and create components
- Apply tokens via Tokens Studio
